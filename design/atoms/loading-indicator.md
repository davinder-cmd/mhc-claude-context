---

## Purpose

Communicates to users that content is loading, processing, or that a change will occur on the page. Prevents users from thinking the system is frozen or unresponsive.

---

## When to Use

- Content is fetching and not yet available
- A user action has been triggered and the system is processing
- Paired with Button's Loading/Processing state for form submissions

## When NOT to Use

- For progress through a defined set of steps — use Progress Indicator instead
