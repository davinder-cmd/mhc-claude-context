## Purpose

Mutually exclusive selection — when only one option from a group can be chosen at a time.

---

## When to Use

- A group of options where exactly one must be selected
- Fewer options (2–5 typically) where showing all options simultaneously aids decision-making

## When NOT to Use

- Multiple selections allowed — use Checkbox
- More than 5–6 options — use Dropdown instead
- Compact space — use Chip Group (Single Choice) as an alternative

---

## Note

Full documentation (variants, sizes, behavior) is in progress in Confluence. Check the link above for the latest.

## Common Mistakes

- Using Radio Buttons for multi-select (use Checkbox)
- Using when there are more than ~6 options (use Dropdown)
