# Range Slider

**Confluence:** https://mobilehealthc.atlassian.net/wiki/spaces/DF/pages/108069096
**Status:** 🚧 Stub — documentation not started

---

## Purpose

Allows users to quickly select a value within a bounded range. The upper and lower bounds are fixed and known.

---

## When to Use

- Selecting a value within a defined, invariable range (e.g., pain scale 1–10, age range filter)
- When approximate values are acceptable and precision isn't critical
- When the range and current value are visually meaningful

## When NOT to Use

- When exact numeric input is needed — use a Text Field with numeric input
- When the range boundaries are unknown or variable
