## Purpose

Allows users to increment or decrement a numeric value in defined steps. Used when precision matters and the range of values is small and bounded.

---

## When to Use

- Adjusting a quantity (e.g., number of servings, days per week)
- When a small, bounded range of discrete values is needed

## When NOT to Use

- Large ranges — use Range Slider instead
- Free-form numeric entry — use a Text Field
