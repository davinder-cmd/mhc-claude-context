3. Responsive Breakpoints

Five breakpoints from mobile (320px) to desktop wide (1600px). Navigation transitions from bottom tabs to top nav at 769px.

XL — 1440px–1600px
Desktop Wide
  • Top nav with full labels + More
  • 2-column body (360px + fluid)
  • Max content width: 1440px centered
  • Generous whitespace (32px margins)
  • Discovery: 2-card grid
  • AI Insights card spans full width
LG — 1025px–1439px
Desktop Standard
  • Top nav with full labels + More
  • 2-column body (340px + fluid)
  • Standard margins (24px)
  • Discovery: 2-card grid
  • Challenge card spans full width
MD — 769px–1024px
Laptop / Small Desktop
  • Top nav with condensed labels + More
  • 2-column body (300px + fluid)
  • Tighter margins (20px)
  • Discovery: 1-card width (stacked)
  • Cards stack vertically in discovery
SM — 481px–768px
Tablet
  • Bottom 5-tab nav
  • 1-column body
  • Cards full-width
  • Profile avatar in header (no hamburger)
  • Rewards bar in body, below welcome
  • Content cards: 2-up grid possible
XS — 320px–480px
Mobile
  • Bottom 5-tab nav
  • 1-column body
  • Cards full-width (100%)
  • Profile avatar replaces hamburger
  • Compact card padding (16px)
  • Tab labels: shortened ("Care" not "Digital Care")

Breakpoint Specifications

Breakpoint Width Range Navigation Layout Margins
XL 1440px – 1600px Top nav (full labels + More) 2-column (360px + fluid) 32px
LG 1025px – 1439px Top nav (full labels + More) 2-column (340px + fluid) 24px
MD 769px – 1024px Top nav (condensed + More) 2-column (300px + fluid) 20px
SM 481px – 768px Bottom tabs (5) 1-column 16px
XS 320px – 480px Bottom tabs (5) 1-column 16px
Key transition at 769px: Navigation moves from bottom tabs to top nav bar. Below 769px = bottom tabs + profile avatar. At/above 769px = top nav + More menu. No hamburger menu at any breakpoint.
Design rationale:
  • 769px transition — Aligns with common tablet breakpoint; gives enough room for 5 tabs + More in top nav
  • 2-column until 768px — Maintains action/discovery split on laptops; collapses to single column on tablets
  • 1600px max — Content doesn't stretch infinitely; centered with whitespace on ultra-wide displays
  • 320px min — Supports older/smaller devices; no content breaks below this width
